1. Скопировать на сервер script_init.sh и папку docker-app
2. Заполнить .env
2. Сделать исполняемым файл script_init.sh командой: chmod +x script_init.sh
3. Запустить файл командой: ./script_init.sh
4. Сделать исполняемым файл docker-app/start.sh командой: chmod +x docker-app/start.sh
5. Запустить файл командой: ./docker-app/start.sh
